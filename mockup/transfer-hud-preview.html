Triển khai icon
3 file PNG family (16/24/32/48/64) hoặc 1 SVG tint runtime qua QPainter. Switch via setHudHint(active, pending, failed) → cache state cuối tránh setIcon thừa.

Anchor logic
QSystemTrayIcon::geometry() trên Win/macOS → align bottom-right popup chạm top-right icon, gap 8px. Linux DE → fallback góc bottom-right availableGeometry.
Compact differences
Row 44px (vs 52 ở mini) · KHÔNG sparkline · KHÔNG ⏸ pause-all (qua right-click menu) · KHÔNG ✕ (click ngoài tự đóng).
Mutual exclusion
Mini đang show + user click tray → mini đóng, popup mở. Không bao giờ có 2 cửa sổ HUD cùng tồn tại.
